How to Edit

1. Click any text on the preview to edit it inline.

2. Use the toolbar to bold, italicize, or change headings.

3. Use Quick Replace to update links and dates globally.

4. Click Copy HTML when done.

5. Paste into your GitHub file to save.

Save to GitHub

1. Copy the HTML from this editor

2. Go to github.com → your bek-portal repo

3. Click the file you edited (e.g. index.html)

4. Click the ✏️ pencil icon to edit

5. Select all (Ctrl+A) and paste

6. Click "Commit changes"

7. Wait ~60 seconds for GitHub Pages to rebuild
